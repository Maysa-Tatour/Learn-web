مقدمة عن JavaScript

JavaScript هي لغة برمجة تضيف تفاعلاً للمواقع، مثل النقر، الحركات، والرسائل.

المتغيرات (Variables)

تُستخدم المتغيرات لتخزين المعلومات مثل الأرقام والنصوص داخل الموقع.

الأحداث (Events)

يمكنك التفاعل مع المستخدم باستخدام أحداث مثل click، hover، وkeyup.

تطبيق عملي

سوف نضيف زر يُظهر رسالة عند النقر عليه باستخدام JavaScript.
