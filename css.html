مقدمة عن CSS

CSS هي لغة تنسيق صفحات الويب، تستخدم لتجميل وتنظيم مظهر المحتوى.

المحددات (Selectors)

تُستخدم لاختيار عناصر HTML لتطبيق التنسيق عليها مثل class و id.

الألوان والخطوط

تعلمي كيفية استخدام الألوان، وتغيير الخطوط لتنسيق مظهر الموقع.

تطبيق عملي

سوف نقوم بإنشاء صفحة بألوان وخطوط مختلفة باستخدام CSS.
